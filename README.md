# flyover_radar
Are you tired of missing out on planespotting cool airplanes? Me too! I've missed one too many Chinooks and Stratolifters so I built this software to alert me when an airplane is approaching.

# Requirements
- Yourself
- ESP32 with WiFi
- Time
- All of the libraries installed (just install every single library available until the errors go away)
- Computer
- Brain

# Config
Everything needs to be configured inside `env.example.cpp` (or your own `.cpp` file with the exact same variables).
